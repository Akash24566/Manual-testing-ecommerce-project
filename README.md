# Manual Testing Project – E-commerce Website

## Project Overview

This project demonstrates end-to-end manual testing of an E-commerce web application.

Website Tested: https://automationexercise.com

## Testing Scope

The following modules were tested:

* User Registration
* Login / Logout
* Product Search
* Product Details
* Cart Management
* Checkout Process
* Contact Form

## Testing Artifacts

* Test Plan
* 30+ Test Cases
* Requirement Traceability Matrix (RTM)
* JIRA-style Bug Reports

## Testing Types

* Functional Testing
* UI Testing
* Regression Testing
* Smoke Testing

## Tools Used

* Microsoft Excel
* GitHub
* Browser DevTools
